# Macro Telegram Bot

Telegram-бот для учета еды и суточной сводки по калориям, углеводам, белкам и жирам.

## Что умеет

- после `/start` и `/add` пошагово спрашивает калории, белки, жиры и углеводы
- после сохранения спрашивает кнопками `Да/Нет`, добавить ли еще один прием пищи
- по-прежнему принимает сообщения вроде `650 ккал, 55 углеводов, 35 белков, 20 жиров`
- сохраняет записи в SQLite
- по команде `/today` отправляет итог за текущий день
- в `00:00` по часовому поясу `BOT_TIMEZONE` отправляет итог за предыдущий день

## Быстрый запуск

```bash
cd /Users/yegorsviergun/Новая\ папка/macro-telegram-bot
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
export TELEGRAM_BOT_TOKEN="токен_от_BotFather"
export BOT_TIMEZONE="Europe/Moscow"
python3 -m macro_bot
```

## Команды

- `/start` - начать пошаговый ввод первого приема пищи
- `/add` - добавить новый прием пищи пошагово
- `/help` - подсказка по формату
- `/today` - текущий итог за день
- `/cancel` - отменить текущий пошаговый ввод

## Переменные окружения

- `TELEGRAM_BOT_TOKEN` - обязателен
- `BOT_TIMEZONE` - часовой пояс для границы суток и отправки в полночь, по умолчанию `Europe/Moscow`
- `MACRO_BOT_DB` - путь к SQLite-базе, по умолчанию `data/macro_bot.sqlite3`

## Примеры сообщений

углеводы 40, белки 30, жиры 10, ккал 410
